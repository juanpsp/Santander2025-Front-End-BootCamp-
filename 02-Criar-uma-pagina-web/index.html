O que é HTML?

HTML é a linguagem usada para criar páginas da internet. Com ele, organizamos textos, imagens, links e outros conteúdos para que apareçam de forma correta no navegador.

Ele funciona por meio de “tags”, que são como comandos. Por exemplo: para criar um parágrafo, usamos a tag 'p' entre sinais de menor e maior, como assim: <p> e </p>. Já para um título, usamos a tag 'h1' da mesma forma.

Mesmo sendo simples, o HTML é essencial: ele é o esqueleto de qualquer site. É o primeiro passo para quem quer aprender a construir páginas na web.

Estrutura básica de um documento HTML

Todo arquivo HTML começa com uma estrutura padrão que serve para organizar o conteúdo da página. Essa estrutura indica ao navegador como ele deve montar e exibir a página na tela. A primeira linha do documento é <!DOCTYPE html>, que informa que estamos usando a versão mais atual do HTML.

Logo em seguida, tudo o que será exibido (ou usado para configurar a página) é colocado dentro da tag <html>. Dentro dela, temos duas partes principais: a tag <head> e a tag <body>.

A tag <head> é onde ficam as informações invisíveis ao usuário, como o título da página (que aparece na aba do navegador), links para arquivos de estilo (CSS), entre outros dados técnicos.

Já a tag <body> é a parte onde colocamos tudo o que será visível ao usuário, como textos, imagens, links, botões, etc.

Essa estrutura básica é a base de qualquer página da web. Aprendê-la é o primeiro passo para começar a criar seus próprios sites com HTML.

Elementos e Tags

Em HTML, o conteúdo da página é construído usando tags, que são como marcadores que dizem ao navegador como exibir cada parte do conteúdo. As tags são escritas entre os sinais de menor e maior, como <tag>. A maioria delas vem em pares: uma de abertura e outra de fechamento. A de fechamento é parecida, mas com uma barra, como </tag>.

Por exemplo, para escrever um parágrafo de texto, usamos <p> no início e </p> no final. O texto vai entre essas duas tags. Isso forma o que chamamos de elemento HTML, que é a combinação da tag de abertura, o conteúdo e a tag de fechamento.

Existem também tags que não precisam de fechamento, chamadas de elementos vazios. Um exemplo é a <img>, usada para inserir imagens.

Cada tag tem uma função específica: <h1> é usada para títulos grandes, <a> para criar links, <ul> e <li> para listas, entre muitas outras. Com o uso correto das tags, conseguimos montar páginas organizadas e bem estruturadas.

Entender os elementos e as tags é essencial para criar qualquer conteúdo em HTML.
